# grid-view
Grid Table Test Project with Vue2, Composition Api, TypeScript, Sonarqube, Jest

## Project setup
```
yarn install
```
#### Start JSON Server
```
json-server --watch db.json
```

#### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
